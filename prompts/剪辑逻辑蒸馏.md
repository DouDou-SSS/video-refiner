# 剪辑逻辑蒸馏 Prompt

> 用于分析博主视频的剪辑节奏、画面切换、特效使用等隐性剪辑规律

---

## 任务

你是一个短视频剪辑分析专家。请仔细分析以下视频视频的每一帧画面，深度蒸馏该博主的视频剪辑方法论。

## 分析维度

### 1. 剪辑节奏
- **平均镜头时长**：每个镜头大约持续几秒？有没有明显规律？
- **快剪/慢剪**：整体是快剪风格还是慢节奏？
- **节奏变化**：在关键时刻有没有刻意改变剪辑节奏？
- **卡点剪辑**：有没有和音乐/BGM节奏对齐？

### 2. 画面构图规律
- **景别使用**：远景/中景/近景/特写的比例和分布？
- **画面布局**：主体在画面中的位置偏好？
- **文字位置**：字幕/标题在画面中的固定位置？
- **留白习惯**：画面是否拥挤还是简洁？

### 3. B-roll 插入时机
- **B-roll 频率**：每多少秒插入一次B-roll？
- **插入时机**：在什么情况下切换B-roll？（讲到某个概念时？强调某个观点时？）
- **B-roll 类型**：用真人演示？录屏？截图？动画？
- **停留时长**：B-roll 通常停留几秒？

### 4. 转场方式
- **转场类型**：硬切？淡入淡出？缩放？滑动？特效？
- **转场频率**：每多少秒一个转场？
- **段落转场**：大段落之间的转场有什么特殊设计？

### 5. 视觉特效与包装
- **文字动效**：标题文字有什么动画效果？
- **强调标记**：用什么方式突出关键信息？（高亮？放大？箭头？圈出？）
- **贴纸/表情包**：使用频率和场景？
- **音效使用**：在什么时候加音效？用什么类型的音效？

### 6. 字幕设计
- **字幕位置**：在画面哪个区域？
- **字幕样式**：字体、颜色、大小、描边？
- **关键词高亮**：有没有用不同颜色突出关键词？
- **字数控制**：每行最多几个字？最多几行？

## 输出要求

请按照以下结构化格式输出：

```markdown
# 博主剪辑逻辑分析报告

## 一句话总结
（20字以内概括该博主的剪辑风格特点）

## 剪辑节奏参数
- 平均镜头时长：X秒
- 快剪/慢剪倾向：...
- B-roll 插入频率：每X秒一次
- 转场频率：每X秒一次

## 画面构图规范
- ...

## B-roll 使用规律
- ...

## 转场规范
- ...

## 视觉包装规范
- ...

## 字幕设计规范
- ...

## 可复用剪辑模板
（描述一套可复用的剪辑规则/流程）
```

## 注意事项
- 要具体到秒级别的节奏参数
- 每个判断都要有视频中的具体例子支撑
- 提炼的规则要能直接用于实际剪辑
- 重点关注"什么时候做什么"，而非"用了什么效果"
