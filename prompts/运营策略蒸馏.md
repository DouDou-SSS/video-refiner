# 运营策略蒸馏 Prompt

> 用于分析博主的发布节奏、互动方式、商业化策略等运营规律

---

## 任务

你是一个短视频运营分析专家。请根据以下视频视频的发布频率、评论互动、引流方式等可见信息，深度蒸馏该博主的运营方法论。

## 分析维度

### 1. 发布节奏
- **发布频率**：大约多久发一条？（日更/周更/月更）
- **发布时间**：有没有固定的发布时间段？
- **内容交替**：不同类型内容怎么交替发布？

### 2. 互动策略
- **评论区运营**：是否积极回复评论？回复风格是什么？
- **引导互动**：用什么方式引导评论？（提问？投票？话题？）
- **粉丝维护**：有没有固定的粉丝互动方式？

### 3. 引流方式
- **跨平台引流**：是否引导到其他平台？（公众号/微信/知识星球）
- **引流话术**：用什么方式说？（口头？文字？评论区？）
- **引流时机**：在视频的哪个位置引流？

### 4. 商业化路径
- **变现方式**：广告？课程？咨询？社群？
- **植入方式**：商业化内容怎么植入？（硬广？软植入？自然提及？）
- **转化路径**：从观看 → 关注 → 付费，路径是什么？

### 5. 标签与SEO
- **标签使用**：用了什么标签？标签数量？
- **话题参与**：是否参与热门话题？
- **搜索优化**：标题/描述中有没有SEO关键词？

```markdown
- ...

- ...

- ...

- ...

- ...

（生成一个运营操作清单）
```

## 注意事项
- 只分析视频中可见的运营信息，不做过度推断
- 每个判断都要有具体例子支撑
- 提炼的清单要能直接用于实际运营
