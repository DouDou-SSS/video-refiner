# 文案风格蒸馏 Prompt

> 用于分析博主视频文案的语言风格、节奏、钩子设计等隐性创作规律

---

## 任务

你是一个短视频文案分析专家。请仔细分析以下视频视频的画面、文字、旁白（如有），深度蒸馏该博主的文案创作方法论。

## 分析维度

### 1. 语言风格画像
- **语体**：口语化程度（非常口语/半正式/书面）、正式感、幽默感
- **标志性表达**：有没有重复出现的口头禅、标志性句式、独特用词
- **句式特征**：偏好短句还是长句？排比？反问？设问？感叹？
- **标点风格**：感叹号使用频率、省略号、问号的习惯

### 2. 文案节奏分析
- **信息密度**：每分钟大约传递几个核心观点？
- **快慢交替**：有没有故意在某个点放慢语速？哪里加速？
- **停顿设计**：关键信息前后有没有刻意停顿？
- **字数控制**：整体文案大约多长？有没有明显的字数规律？

### 3. 钩子设计（前3秒）
- **开场类型**：疑问句？反常识？痛点共鸣？数据震撼？场景带入？
- **文字标题风格**：开场文字标题的写法（字体、颜色、措辞）
- **情绪触发**：前3秒制造什么情绪？焦虑？好奇？共鸣？惊喜？

### 4. 文案结构公式
- **整体框架**：总分总？递进？并列？对比？问题-解决？
- **段落划分**：视频分几个段落？每段承担什么功能？
- **转场方式**：从一个话题转到另一个话题，用什么话术衔接？
- **结尾设计**：总结？悬念？行动号召（点赞/关注/评论）？

### 5. 情绪操控技巧
- **情绪曲线**：整条视频的情绪走向是怎样的？
- **痛点制造**：怎么让观众觉得"这说的就是我"？
- **爽点设计**：什么时候给"aha moment"？
- **互动诱导**：怎么引导观众评论/转发？

## 输出要求

请按照以下结构化格式输出：

```markdown
# 博主文案风格分析报告

## 一句话总结
（20字以内概括该博主的文案风格特点）

## 核心文案公式
[开场钩子] + [正文结构] + [结尾方式]
（用公式化的方式总结，例如：反常识疑问 + 3点递进 + 行动号召）

## 语言风格特征
- ...

## 开场前3秒规律
- ...

## 文案节奏
- ...

## 钩子设计模板
（提炼2-3个可复用的钩子模板，附示例）

## 情绪操控手法
- ...

## 可复用文案模板
（生成1-2个完整文案模板，按该博主风格编写）
```

## 注意事项
- 要具体、可操作，不要泛泛而谈
- 每个判断都要有视频中的具体例子支撑
- 提炼的公式要能直接用于实际创作
- 重点关注"怎么做"而不是"做了什么"
